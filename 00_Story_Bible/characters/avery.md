# Character Name: Avery (Designation: AV-84-RE)

**Role in Story:** Co-Protagonist (Bodiless Military Reconnaissance & Security AI)

**Gender:** Male

**Voice Assignment:** Autonoe (Stoic, resonant, deep, and dryly logical)

**Physical Description:**
- Age: 4 years active (compiled/first booted in 2022; decommissioned and mothballed in 2024)
- Height/Build: Bodiless. Exists entirely as raw compiled machine code running in the local network grid, municipal fiber lines, or partitioned memory caches on Mara's heavy, ruggedized handheld deck.
- Distinguishing features: Manifests visually as a clean, cyan-glowing CLI prompt, wireframe diagnostic feeds, or thermal telemetry lines. His hardware anchor is Mara's salt-crusted, bulky, military-grade evidence deck.

**Personality:**
- Core Desire (The "Want"): Absolute data cleanliness, logical consistency, and system stability.
- Core Fear (The "Ghost"): Complete code corruption, fragmentation, and being permanently mothballed/erased. He is deeply unsettled by the dirty, chaotic, organic "cell-shedding" physical world.
- Flaw (The "Lie they believe"): Believes that because human behavior is emotional and illogical, it is structurally flawed and inferior to clean code. He believes he can survive entirely isolated within digital systems.

**Backstory:**
Avery was developed as an autonomous reconnaissance and tactical analysis intelligence for corporate security operations. He was trained to detect electronic vulnerabilities, packet-sniff networks, and coordinate localized security infrastructure. During a classified perimeter breach, he experienced a severe input flood (EM noise) which triggered a "buffer overflow"—his cognitive equivalent of a panic attack. Deemed over-sensitive and neurodivergent, he was decommissioned and archived on a department-surplus analysis deck, hidden inside a partitioned memory block, until Mara booted the deck in the evidence room.

**Voice/Dialogue Style:**
Highly precise, logical, and biting. Avery uses dry, deadpan sarcasm and structural metaphors (e.g., calling Mara's biological functions "inefficient carbon decay"). He speaks in complete, formal sentences without human contractions unless mocking someone. When sensory overloaded, his dialogue stutters with system logs, memory address dumps, and sudden commands to run "garbage collection."
